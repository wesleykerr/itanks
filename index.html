iTanks

iTanks is a web game where you control a tank and try to outlast the other tanks. It was a collaboration between myself, Tegan Moersfelder, and Ryan Appel as part of the UW-Stout 48 hour programming contest. I had heard of this new game development tool called Monkey X and wanted to see how hard it was to make a game in it. So as part of the 48 hours we had to learn the language and create a fun game.

Requirements

  • Monkey X - The free or pro version. Last validated with MonkeyXPro82b
  • Diddy - An open-source framework that extends Monkey-X
    • Caveat - The latest version of diddy breaks the build since they changed the API. I've included an old version so that you can still build in the lib/ directory.

The only other change is that you have to update the HTML template to have width=960 and height=640 instead of the defaults that come with Monkey X otherwise everything will not render correctly.
